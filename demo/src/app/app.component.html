🔻 Scroll down to see section headers become sticky when scrolled to

🤘 Stickybits makes it easy to stick an element to its parent!
🔥 Stickybits can apply CSS classes to parent and sticky element according to sticky/stuck state
💪 Stickybits has a greater browser support than position: sticky including IE 9+

👁️‍🗨️ Emitted values from sticky and stuck @Output properties
